# Bank Customer Classification using Machine Learning

## Project Overview
This project predicts customer behavior in a banking dataset using machine learning classification algorithms.

## Problem Statement
Banks want to predict whether a customer will subscribe to a product or service based on historical data.

## Dataset
The dataset contains customer information such as:
- Age
- Job
- Balance
- Marital status
- Loan information

## Technologies Used
Python  
Pandas  
NumPy  
Matplotlib  
Seaborn  
Scikit-learn  

## Machine Learning Models
- Logistic Regression
- Random Forest
- Decision Tree
- Support Vector Machine

## Project Workflow
1. Data Collection  
2. Data Cleaning  
3. Exploratory Data Analysis  
4. Feature Engineering  
5. Model Training  
6. Model Evaluation  

## Results
Random Forest performed best with the highest accuracy.
